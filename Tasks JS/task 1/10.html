10 Напишите функцию sumTo(n), которая для данного n вычисляет сумму чисел от 1 до n Сделайте три варианта решения: 1. С использованием цикла. 2. Через рекурсию, т.к. sumTo(n) = n + sumTo(n-1) для n > 1. 3. С использованием формулы для суммы арифметической прогрессии из алгебры.

Можно ли при помощи рекурсии посчитать sumTo(100000)? Если нет, то почему? Ответ : Максимальная глубина рекурсии ограничена и составляет около 10000, но это число зависит от браузера и может быть в 10 раз меньше.
Какой вариант решения самый быстрый? Самый медленный? Почему? Ответ : решение по формуле работает быстрее всех, использует всего три операции для любого n. с циклом — быстрее рекурсии, так как операции по сути те же, но нет накладных расходов на вложенный вызов функции.
